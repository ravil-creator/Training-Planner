# Training Planner - План тренировок

**Автор:** Равиль

## Описание

Приложение для планирования тренировок с графическим интерфейсом на Tkinter.

## Возможности

- ➕ Добавление тренировок (дата, тип, длительность)
- 🔍 Фильтрация по типу и дате
- 🗑️ Удаление записей
- 💾 Автоматическое сохранение в JSON

## Запуск программы

```bash
python training_planner.py
